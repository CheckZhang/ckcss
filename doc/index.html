How to name a CSS Class in Object-Oriented CKCSS?

In CKCSS, we use below naming format:

  • ".wordWord": Base Class name
  • ".wd_word" : Child Class name,the letters before underline is the abbreviation of it's parent class,and it aways follow it's parent Class in css,eg:".parentClass.pc_ChildClass;"
  • "wd-word" : For the Child Dom Class
  • "WordWord": Dom ID's Css
